# .NET Movie App

A simple ASP.NET Core MVC web application for managing a movie collection. The app demonstrates building a **data-driven website** using **Entity Framework Core** and **SQLite**, featuring dynamic search, filtering, and CRUD operations.

## Features
- View a list of movies with details: title, genre, release date, price, and rating  
- Search movies by title and filter by genre or release year  
- Add, edit, and delete movies through an intuitive interface  
- Responsive design optimized for desktop and mobile devices  

## Technologies
- ASP.NET Core MVC  
- Entity Framework Core  
- SQLite database  
- HTML, CSS, C#
